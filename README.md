# 🐍 Exercícios de Condicionais em Python
Este repositório contém um conjunto de exercícios introdutórios em Python, focados no uso de estruturas condicionais (if, elif, else).
Os exercícios incluem verificações de números, idades, autenticação simples, cálculos e classificações.  

---

# 📘 Conteúdos abordados
Verificação de números positivos/negativos

Determinação de maioridade

Verificação de paridade

Comparação de valores

Validação de password

Classificação de notas

Classificação por faixa etária

Verificação de múltiplos

Autenticação simples (user + senha)

Verificação de intervalos

Simulação de levantamento bancário

Determinação do maior entre quatro valores

Cálculo de IMC

Cálculo de descontos

Verificação de ano bissexto
